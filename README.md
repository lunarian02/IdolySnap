# IdolySnap

A virtual photo diary inspired by idol photocards and daily memories.

IdolySnap transforms everyday moments into collectible memories through AI-generated characters and personalized photo card experiences.

## Concept

Most photo apps focus on storing memories.

IdolySnap focuses on creating emotional connections by combining daily journaling, virtual companions, and collectible photo cards.

Inspired by:

* K-pop photocards
* Daily journaling
* 1 Second Everyday
* Virtual companion experiences

## Features

* Daily photo entries
* AI-generated character interactions
* Virtual photocard collection
* Personal memory timeline

## Vision

Turn everyday moments into collectible stories.
